---

## What is the SDS?

The SDS (Sprachstandstest Deutsch Schweiz) is the standardized German language test used in Switzerland to assess language proficiency for residence permits and naturalization. It is **adaptive** — the test adjusts difficulty based on your answers.

### Key Facts

| Aspect | Detail |
|--------|--------|
| **Skills tested** | Reading, Listening, Writing, Speaking |
| **Levels** | pre-A1 → B2 (adaptive) |
| **Speaking format** | Live interview (phone/video call during test) |
| **Language options** | Standard German (Hochdeutsch) or Swiss German (Mundart) for speaking |
| **Test medium** | Tablet / computer |
| **Adaptive scoring** | Correct answers → harder questions; mistakes early can cap your level |

## 6-Week Plan at a Glance

| Week | Focus | Key Activities |
|------|-------|---------------|
| **1** | Formats + Baseline + Vocab 1–3 | Learn all task types; vocab: Work, Job Search, Authorities |
| **2** | A1/A2 Writing + Listening Basics | Text messages, simple emails; picture-matching listening |
| **3** | Reading Focus + Speaking Intro | Job ads, notices; self-introduction + process descriptions |
| **4** | B1 Writing + Longer Comprehension | Inquiry emails; extended listening/reading; interview practice |
| **5** | B2 Argumentation + Timed Practice | Pro/contra writing & speaking; mixed timed drills |
| **6** | Mock Tests + Weak-Area Repair | 2–3 full simulations; error log; targeted fixes |

### Daily Minimum: 45–75 minutes

- **15 min** — Vocabulary (one topic)
- **15–20 min** — Listening or Reading
- **15–20 min** — Writing or Speaking (alternate days)

## How to Use This Repo

1. **Start** with `docs/00-todo/master-checklist.md` — your single source of truth for progress
2. **Learn the formats** in `docs/01-exam-formats/` before any practice
3. **Build vocabulary** from `docs/02-vocabulary/` — do 1 pack every 3–4 days
4. **Train skills** using `docs/03-skills/` — strategies first, then practice exercises
5. **Follow the weekly plan** in `docs/04-weekly-plan/`
6. **Reference grammar** in `docs/05-grammar/` as needed
7. **Simulate the test** using `docs/06-test-day/mock-test-checklist.md`
8. **Serve locally** with `uv run mkdocs serve`

## SDS Official Thematic Spectrum (10 Domains)

All vocabulary and practice material covers these domains:

1. **Arbeit** — Work conditions, communication at work
2. **Jobsuche** — Job ads, applications, CVs
3. **Behörden** — Authorities, advisory offices (Beratungsstellen/Ämter)
4. **Einkaufen** — Shopping, ads, online shopping
5. **Gesundheit** — Doctor visits, healthy eating, pharmacy
6. **Kinder** — Childcare, school, parent-teacher communication
7. **Medien & Freizeit** — Hobbies, travel, media consumption
8. **Post / Bank / Versicherung** — Health insurance, e-banking, postal services
9. **Verkehr** — Public transport, mobility, driving
10. **Wohnen** — Apartment search, neighbors, neighborhood
