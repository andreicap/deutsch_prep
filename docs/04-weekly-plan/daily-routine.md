# Daily Routine — 45–75 Minutes

> Use this as a daily template. Pick one activity from each row.

---

## Structure

| Block | Duration | Focus | Options |
|-------|----------|-------|---------|
| **1** | 15 min | Vocabulary | Study 1 new pack OR review an old one (self-test) |
| **2** | 15–20 min | Input skill | Listening drill OR Reading exercise (alternate days) |
| **3** | 15–20 min | Output skill | Writing practice OR Speaking practice (alternate days) |

---

## Quick Activity Ideas

### Vocabulary (15 min)
- Study a new vocab pack from `02-vocabulary/`
- Self-test: cover English column, translate German → English
- Reverse: cover German column, translate English → German
- Write 5 sentences using new words
- Make flashcards (physical or Anki)

### Listening (15–20 min)
- Number dictation drill
- One-pass summary (1–2 min audio clip)
- Short dialogue comprehension
- Announcement practice
- Speaker tracking (interview audio)

### Reading (15–20 min)
- True/false exercise on a job ad
- Title-matching exercise
- Multiple choice (timed: 5 min)
- Read a real article from SRF or 20min.ch
- Form-filling exercise

### Writing (15–20 min)
- Write 1 email from practice prompts
- Reply to 2 text messages
- Write a B2 opinion paragraph (1 argument for + 1 against)
- Practice typing on tablet keyboard
- Rewrite an old email — try to improve it

### Speaking (15–20 min)
- Self-introduction (record, listen, improve)
- Typical day (record, listen, improve)
- Process description (pick one, record)
- Pro/contra on a topic (2 min prep, then record)
- Answer 5 random personal questions out loud
