# Speaking Strategies for SDS

## Core Principle: Structure Over Perfection

The examiner is listening for **structured, understandable speech** — not perfect grammar. A well-organized answer with some grammar mistakes scores higher than a grammatically perfect one-word answer.

---

## Strategy 1: Memorize Structures, Not Scripts

Don't learn answers word-for-word. Instead, learn **frameworks** you can fill with any content:

| Task | Framework |
|------|-----------|
| Personal question | Topic sentence → 2–3 details → follow-up comment |
| Process description | Zuerst → Dann → Danach → Zum Schluss |
| Pro/contra | Thema → 2 Vorteile → 2 Nachteile → Meine Meinung |
| Picture description | Was ich sehe → Was passiert → Was ich denke |

## Strategy 2: Expand Your Answers

Never give one-word answers. Use the **3-sentence rule**:

| Question | Bad answer | Good answer |
|----------|-----------|-------------|
| Was machen Sie beruflich? | "Ich arbeite." | "Ich arbeite als Reinigungskraft in einem Hotel. Ich arbeite dort seit 3 Jahren. Die Arbeit gefällt mir, weil die Kollegen nett sind." |
| Haben Sie Kinder? | "Ja, zwei." | "Ja, ich habe zwei Kinder. Mein Sohn ist 8 und geht in die Primarschule. Meine Tochter ist 5 und geht in den Kindergarten." |

## Strategy 3: Use Fillers and Stalling Phrases

When you need time to think:

| Deutsch | Purpose |
|---------|---------|
| Also… | Well… (most common filler) |
| Ja, genau… | Yes, exactly… |
| Das ist eine gute Frage. | That's a good question. |
| Lassen Sie mich kurz überlegen. | Let me think for a moment. |
| Wie soll ich sagen… | How should I say… |
| Ich meine damit, dass… | What I mean by that is… |

These are **not** signs of weakness — native speakers use them constantly.

## Strategy 4: Self-Correct Confidently

If you make a mistake, correct it smoothly:

- *"Ich gehe zum Arzt… nein, ich meine, ich bin zum Arzt gegangen."*
- *"Er arbeitet… also, sie arbeitet bei Migros."*

Self-correction shows awareness and is scored positively.

## Strategy 5: Handle Follow-Up Questions

At higher levels, the examiner asks probing questions. Be ready:

| Examiner says | You can respond with |
|---------------|---------------------|
| Können Sie das genauer erklären? | Ja, ich meine damit, dass… / Zum Beispiel… |
| Warum denken Sie das? | Der Grund ist, dass… / Weil… |
| Gibt es noch andere Gründe? | Ja, ausserdem… / Ein weiterer Grund ist… |
| Was wäre die Alternative? | Eine Möglichkeit wäre, dass… |
| Sind Sie sicher? | Ja, ich denke schon, weil… / Vielleicht nicht ganz, aber… |

## Strategy 6: Ask for Repetition (It's OK!)

If you don't understand:
- *Entschuldigung, können Sie die Frage bitte wiederholen?*
- *Ich habe das nicht ganz verstanden. Können Sie das bitte noch einmal sagen?*
- *Was meinen Sie genau mit…?*

Asking for repetition is much better than guessing wrong.

## Speaking Warm-Up Routine (5 min before practice)

1. Count from 1 to 20 out loud
2. Say the days of the week and months
3. Say 5 sentences about your day: *Heute bin ich um 7 Uhr aufgestanden. Dann habe ich…*
4. Name 5 objects you can see and describe them

---

## Practice Method: Record and Review

1. Set a timer (1–3 minutes depending on task)
2. Record yourself on your phone
3. Listen back and note:
   - Did I answer the question completely?
   - Was my structure clear?
   - Where did I hesitate too long?
   - Which words did I forget?
4. Try again, aiming to fix 1–2 issues
5. Repeat 3 times per topic

## Common Speaking Topics (from test-taker reports)

- [ ] Introduce yourself (name, origin, family, job, hobbies)
- [ ] Talk about your family / job + follow-up questions
- [ ] Describe a typical day at work
- [ ] Your favorite food / cooking
- [ ] How you learn German
- [ ] B2: Should shops be open on Sundays? (pro/contra)
- [ ] B2: Home office advantages/disadvantages
- [ ] Describe: how to go to the doctor / search for an apartment / extend a permit
