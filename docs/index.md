# SDS Exam Preparation — Deutsch Prep

> **Sprachstandstest Deutsch Schweiz (SDS)** — Comprehensive preparation system for all 4 skills at levels pre-A1 through B2.

---

## Master Checklist

Track your progress across all preparation areas. Check items off as you complete them.

### Phase 0: Setup & Orientation

- [ ] Read the full `README.md` for the 6-week plan overview
- [ ] Review all 4 exam format guides (`01-exam-formats/`)
- [ ] Do 1 sample task for each skill (reading, listening, writing, speaking)
- [ ] Assess your current level honestly (pre-A1 / A1 / A2 / B1 / B2)
- [ ] Set your target level (minimum needed for your permit/application)
- [ ] Practice typing on a tablet/touch keyboard (test is on a tablet!)

### Phase 2: Skill Training — Reading

- [ ] Read [Reading Strategies](03-skills/reading/strategies.md)
- [ ] Practice: Word ↔ Picture matching (pre-A1/A1)
- [ ] Practice: Job ad true/false (A2)
- [ ] Practice: Title ↔ text matching (B1)
- [ ] Practice: Multiple choice — 2 options (A1–A2)
- [ ] Practice: Multiple choice — 3 options (B2)
- [ ] Do 5 timed reading exercises (ads, notices, short articles)
- [ ] Do 5 timed true/false exercises on job ads
- [ ] Do 3 title-matching exercises

### Phase 3: Skill Training — Listening

- [ ] Read [Listening Strategies](03-skills/listening/strategies.md)
- [ ] Practice: 3 words → match 3 pictures (pre-A1/A1)
- [ ] Practice: Short text → correct picture per line (A1/A2)
- [ ] Practice: Survey/interview → assign people to statements (B1)
- [ ] Practice: Survey → satisfied / dissatisfied / no statement (B1/B2)
- [ ] Do 10 "one-pass listening" drills (no replay)
- [ ] Practice with workplace dialogues
- [ ] Practice with short announcements (train, shop, office)

### Phase 4: Skill Training — Writing

- [ ] Read [Writing Strategies](03-skills/writing/strategies.md)
- [ ] Read [Email Templates](03-skills/writing/email-templates.md)
- [ ] Practice: Fill missing letters / spelling (pre-A1)
- [ ] Practice: Copy details into form fields (pre-A1/A1)
- [ ] Practice: Reply to text messages (A1)
- [ ] Write 10 simple emails — appointment, cancellation, request (A2)
- [ ] Write 10 job-related emails — holiday request, interview reschedule (B1)
- [ ] Write 5 inquiry/information-request emails (B1)
- [ ] Write 5 opinion/pro-con texts (~200 words) (B2)
- [ ] Practice writing on a tablet/touch keyboard

### Phase 5: Skill Training — Speaking

- [ ] Prepare 60–90 sec self-introduction (memorize structure, not script)
- [ ] Prepare 2–3 min "typical day" (work + free time)
- [ ] Prepare 2–4 min process descriptions (3 different ones)
- [ ] Prepare B2 pro/contra + opinion structure
- [ ] Practice self-introduction out loud 5× (record yourself)
- [ ] Practice "typical day" out loud 5× (record yourself)
- [ ] Practice 3 different process descriptions out loud
- [ ] Practice 3 pro/contra topics (Sunday shopping, home office, public transport)
- [ ] Do at least 2 mock speaking interviews (with a partner or tutor)
- [ ] Practice handling follow-up / counter-questions

### Phase 6: Grammar & Connectors

- [ ] Learn B1/B2 connectors (weil, obwohl, deswegen, außerdem)
- [ ] Learn formal + informal email phrases
- [ ] Know when to use which register
- [ ] Practice using connectors in writing
- [ ] Practice email openings/closings from memory

### Phase 7: Mock Tests & Final Review

- [ ] Complete Mock Test 1 (all 4 skills, timed)
- [ ] Complete Mock Test 2 (all 4 skills, timed)
- [ ] Complete Mock Test 3 (all 4 skills, timed)
- [ ] Review error log — identify top 3 weak areas
- [ ] Targeted repair: drill weak areas for 2–3 days
- [ ] Final vocabulary review (all 10 packs)
- [ ] Final email writing review (write 3 emails under time pressure)
- [ ] Final speaking rehearsal (all modules once)

---

## Daily Minimum (45–75 min)

- **15 min** — Vocabulary (one topic)
- **15–20 min** — Listening or Reading
- **15–20 min** — Writing or Speaking (alternate days)
