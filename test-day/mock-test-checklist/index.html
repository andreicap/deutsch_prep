Use this to run a full practice test under realistic conditions. Do 2–3 mock tests in Week 6.

Setup#

  • Set aside 90–120 minutes (uninterrupted)
  • Use a tablet or computer (not paper)
  • No dictionary, no notes during the test
  • Set a timer for each section
  • Have a recording device ready for speaking

Mock Test Structure#

Task Time Source
Word ↔ picture matching (5 items) 3 min Create from vocab packs or skip if above A1
Job ad true/false (6 items) 7 min 03-skills/reading/practice-exercises.md Ex.1 or find a real ad

Reading score: ___ / 21

Section 2: Listening (25 minutes)#

Task Time Source
3 words → match pictures 3 min YouTube: “Deutsch hören A1”
Short text → correct picture 5 min DW Nico’s Weg or similar
Interview → assign statements (3 speakers) 8 min Easy German interview or DW Top-Thema
Survey → satisfied/dissatisfied (4 items) 9 min DW or SRF podcast

Important: Play audio only ONCE for B1/B2 tasks.

Listening score: ___ / 13 (estimate based on correct answers)

Section 3: Writing (30 minutes)#

Task Time Prompt
Fill form fields 3 min 03-skills/reading/practice-exercises.md Ex.5
Reply to text message 3 min Pick any from 03-skills/writing/practice-prompts.md (1–5)
Write a simple email (A2) 8 min Pick from Prompts 6–15
Write a B1 inquiry email 8 min Pick from Prompts 16–25
Write a B2 opinion text (~200 words) 8 min Pick from Prompts 26–30

Writing self-assessment:

Criteria Score (1–5)
Answered all parts of each prompt
Correct greeting + sign-off
Used connectors (weil, deshalb, ausserdem…)
Spelling and capitalization
Formal/informal register correct
B2 text has clear pro/contra + opinion

Section 4: Speaking (15 minutes)#

Record everything. Have someone ask you questions, or use a timer.

Task Time Prompt
Self-introduction 1–2 min Stellen Sie sich vor.
Personal questions (3 random) 3 min See list below
Process description 3 min Beschreiben Sie, wie man [X] macht.
Pro/contra + opinion 4 min Was denken Sie über [topic]?

Random personal questions (pick 3):

  1. Was machen Sie beruflich? (Data Scientist bei SMG)
  2. Wie sieht ein typischer Tag bei Ihnen aus? (Tram, Büro, Daten, Sport)
  3. Was machen Sie in der Freizeit? (Volleyball, Fussball, Schach, Skifahren, Restaurants)
  4. Was ist Ihr Lieblingsessen? (moldauisch, italienisch — aus Studienzeit in Italien)
  5. Wie haben Sie Deutsch gelernt? (Kurs in Zürich, Selbststudium)
  6. Was gefällt Ihnen an der Schweiz? (Berge, Lebensqualität, Sicherheit)
  7. Woher kommen Sie? (Chișinău, Moldau — studiert in Moldau, Italien, Schweden)
  8. Wie kommen Sie zur Arbeit? (Tram, ca. 15 Minuten)

Speaking self-assessment:

Criteria Score (1–5)
Spoke in full sentences (not one-word)
Used structures (zuerst, dann, zum Schluss)
Answered follow-up questions
Pro/contra was balanced with clear opinion
Spoke for the full time (didn’t stop too early)
Sounded natural (not like reading a script)
