## Verbs

| Deutsch | English | Example |
|---------|---------|---------|
| kaufen | to buy | Ich kaufe Brot und Milch. |
| einkaufen | to go shopping | Ich gehe am Samstag einkaufen. |
| verkaufen | to sell | Er verkauft Gemüse auf dem Markt. |
| bestellen | to order | Ich bestelle das online. |
| bezahlen | to pay | Kann ich mit Karte bezahlen? |
| kosten | to cost | Wie viel kostet das? |
| umtauschen | to exchange | Kann ich das umtauschen? |
| zurückgeben | to return | Ich möchte das zurückgeben. |
| sparen | to save (money) | Bei der Aktion kann man sparen. |
| liefern | to deliver | Wann wird es geliefert? |
| probieren | to try (on/out) | Kann ich das probieren? |
| suchen | to look for | Ich suche eine Winterjacke. |

## Useful Phrases

| Deutsch | English |
|---------|---------|
| Wie viel kostet das? | How much does this cost? |
| Haben Sie das in einer anderen Grösse? | Do you have this in a different size? |
| Kann ich mit Karte bezahlen? | Can I pay by card? |
| Ich hätte gerne… | I would like… |
| Wo finde ich…? | Where can I find…? |
| Das ist zu teuer. | That's too expensive. |
| Gibt es einen Rabatt? | Is there a discount? |
| Ich möchte das umtauschen. | I'd like to exchange this. |
| Die Lieferung dauert 3 Tage. | Delivery takes 3 days. |
| Ich habe online bestellt. | I ordered online. |
| Kann ich die Quittung haben? | Can I have the receipt? |
| Das passt nicht. | It doesn't fit. |
| Ist das im Angebot? | Is that on offer? |
