Speaking Module: Pro/Contra + Opinion (Argumentation)

Duration: 2–4 minutes. B2-level task. Present both sides of a topic and give your opinion. Structure (memorize this framework) 1. INTRODUCTION (10 sec) "Es geht um die Frage, ob… / Das Thema [X] wird oft diskutiert." 2. ARGUMENTS FOR — 2 points (45–60 sec) "Einerseits… / Ein Vorteil ist, dass… / Dafür spricht, dass…" + Example or explanation for each 3. ARGUMENTS AGAINST — 2 points (45–60 sec) "Andererseits… / Ein Nachteil ist, dass… / Dagegen spricht, dass…" + Example or explanation for each 4. YOUR OPINION (20–30 sec) "Meiner Meinung nach… / Ich persönlich denke, dass…" + One reason why 5. CONCLUSION (10 sec) "Zusammenfassend würde ich sagen, dass…" Key Phrases for Argumentation Introducing the topic Es geht um die Frage, ob… Das Thema… wird oft diskutiert. Viele Menschen fragen sich, ob… Heutzutage ist… ein wichtiges Thema. Arguments FOR Einerseits… Ein Vorteil ist, dass… Dafür spricht, dass… Ein wichtiger Punkt ist, dass… Ausserdem / Darüber hinaus… Zum Beispiel… Arguments AGAINST Andererseits… Ein Nachteil ist, dass… Dagegen spricht, dass… Man darf aber nicht vergessen, dass… Ein Problem dabei ist, dass… Allerdings… Your opinion Meiner Meinung nach… Ich persönlich denke, dass… Ich bin der Meinung, dass… Aus meiner Sicht… Ich finde, dass… Conclusion Zusammenfassend kann man sagen, dass… Alles in allem denke ich, dass… Insgesamt überwiegen die Vorteile / Nachteile. Speaking Module: Process Descriptions (Ablauf beschreiben)

Duration: 2–4 minutes. Common at B1 level. Describe the steps of a familiar process. Structure (use for ANY process) Introduction: "Wenn man [X] machen möchte, muss man Folgendes tun:" Step 1: "Zuerst..." Step 2: "Dann..." Step 3: "Danach..." Step 4: "Anschliessend..." Step 5: "Zum Schluss..." Closing: "So funktioniert das normalerweise." Speaking Strategies for SDS

Core Principle: Structure Over Perfection The examiner is listening for structured, understandable speech — not perfect grammar. A well-organized answer with some grammar mistakes scores higher than a grammatically perfect one-word answer. Strategy 1: Memorize Structures, Not Scripts Don’t learn answers word-for-word. Instead, learn frameworks you can fill with any content: Task Framework Personal question Topic sentence → 2–3 details → follow-up comment Process description Zuerst → Dann → Danach → Zum Schluss Pro/contra Thema → 2 Vorteile → 2 Nachteile → Meine Meinung Picture description Was ich sehe → Was passiert → Was ich denke Strategy 2: Expand Your Answers Never give one-word answers.
